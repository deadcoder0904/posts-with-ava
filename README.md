# posts-with-ava

# Installation & Configuration

(1) Clone the project

`git clone git@github.com:deadcoder0904/posts-with-ava.git`

(2) Install the dependencies

`npm i`  or `npm install`

(3) Run development server while developing

`npm run dev`

(4) Start the application

`npm start`

(5) Run the tests

`npm t`  or `npm test`
